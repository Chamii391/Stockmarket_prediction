# AAPL LSTM Stock Price Predictor (Flask UI)

This project trains an LSTM model on Apple (AAPL) stock **Open** prices and deploys it using **Flask** with a simple web UI.  
User enters the last **50 open prices**, and the app predicts the next open price.

---

## Features
- LSTM model for next-day Open price prediction
- MinMaxScaler normalization
- Flask backend API
- Clean HTML/CSS UI
